# Portofolio Rahmat Akmal

Website portofolio pribadi built with React + Vite + Tailwind CSS.

🔗 **Live:** https://rahmat-akmal.vercel.app/

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Build for production:
   `npm run build`
